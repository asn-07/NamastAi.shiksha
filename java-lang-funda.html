1. Identifiers

A name in Java program is called identifier. It may be class name, method name, variable name, or label name.

Rules for Java Identifiers:

  1. Allowed characters: a-z, A-Z, 0-9, underscore (_), dollar ($)
  2. Cannot start with digit: ABC123 (valid) but 123ABC (invalid)
  3. Case sensitive: number, Number, and NUMBER are different identifiers
  4. No length limit: But recommended to keep under 15 characters
  5. Cannot use reserved words: int if = 10; (invalid)
  6. Can use predefined class names: int String = 10; (valid but not recommended)
public class Test {
  public static void main(String[] args) {
    int total_number = 100; // valid
    int Total# = 50; // invalid - contains #
    int 123abc = 25; // invalid - starts with digit
  }
}

2. Reserved Words

Reserved identifiers that associate specific functionality or meaning in Java.

Category Keywords
Data Types (8) byte, short, int, long, float, double, char, boolean
Flow Control (11) if, else, switch, case, default, for, do, while, break, continue, return
Modifiers (11) public, private, protected, static, final, abstract, synchronized, native, strictfp, transient, volatile
Exception Handling (6) try, catch, finally, throw, throws, assert
Class Related (6) class, package, import, extends, implements, interface
Object Related (4) new, instanceof, super, this
Note: All reserved words contain only lowercase alphabet symbols. Java has keywords like enum (1.5v), assert (1.4v), and strictfp (1.2v).
Unused Keywords: goto and const are reserved but unused in Java. Using them causes compile-time errors.
